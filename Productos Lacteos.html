Productos Lacteos

Mantequilla: - Ingredientes: crema de leche, sal (opcional) - Proceso: se bate la crema hasta que se separe la mantequilla del suero, luego se lava y se amasa para eliminar exceso de líquido. Queso Panela: - Ingredientes: leche pasteurizada, cuajo, sal - Proceso: se calienta la leche, se agrega cuajo y se deja coagular, luego se coloca la cuajada en moldes y se prensa para darle forma. Queso Fresco: - Ingredientes: leche pasteurizada, cuajo, sal, bacterias lácticas (opcional) - Proceso: se calienta la leche, se agrega cuajo y se deja coagular, luego se corta la cuajada y se deja escurrir. Queso Botanero: - Ingredientes: leche cruda de vaca o cabra, cuajo, sal, bacterias lácticas (opcional) - Proceso: se calienta la leche, se agrega cuajo y se deja coagular, luego se corta la cuajada, se prensa y se deja madurar. .
